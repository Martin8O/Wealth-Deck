# Wealth Deck

Modern personal finance and portfolio dashboard focused on clarity, speed, and actionable insights.

## Overview

Wealth Deck is a clean and responsive wealth-tracking web application designed to help users monitor investments, portfolio allocation, asset performance, and overall financial overview in one place.

Built with a modern frontend stack and optimized UX, the project focuses on:

* Fast and intuitive user experience
* Clear portfolio visualization
* Modern dashboard design
* Responsive layout across devices
* Scalable architecture for future expansion

Live Demo: [https://wealthdeck.lovable.app/](https://wealthdeck.lovable.app/)

Repository: [https://github.com/Martin8O/Wealth-Deck](https://github.com/Martin8O/Wealth-Deck)

# Getting Started

## Prerequisites

Make sure you have installed:

* Node.js
* npm or yarn

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Martin8O/Wealth-Deck.git
```

Navigate into the project:

```bash
cd Wealth-Deck
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

# Build for Production

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```
